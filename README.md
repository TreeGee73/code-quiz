## Description

The purpose of this project is to create a quiz that utilizes JavaScript to manage what is displayed by interfacing with the DOM rather than making changes to the HTML. This is a good way to avoid "repeating yourself" when creating a dynamic page. This method listens for events like mouse clicks and responds to the events, as well as allows for the creation of elements, manipulation of elements (changing attributes, adding content, etc.), and creating a locally stored record of data based on form input.

### Project Criteria

AS A coding boot camp student  
I WANT to take a timed quiz on JavaScript fundamentals that stores high scores  
SO THAT I can gauge my progress compared to my peers

GIVEN I am taking a code quiz  
WHEN I click the start button  
- [x] A timer starts and I am presented with a question  
WHEN I answer a question  
- [x] I am presented with another question  
WHEN I answer a question incorrectly  
- [x] Time is subtracted from the clock  
WHEN all questions are answered or the timer reaches 0  
- [x] The game is over  
WHEN the game is over  
- [x] I can save my initials and score

## Usage

This is an interactive JavaScript fundamentals quiz application featuring:
- Timed quiz with countdown functionality
- Dynamic question and answer display
- Time penalty for incorrect answers
- Score tracking based on remaining time
- Local storage of high scores with player initials
- Ability to view, clear, and restart quiz

**Live Application:** https://treegee73.github.io/code-quiz/
